# Repo for the systematic review on NLP/LLM tools for Dutch EHR text

All code, screened, included and excluded references for the article titled 'Natural language processing and language models for Dutch clinical text: a systematic review' (currently under submission), for which an earlier version was preprinted at: https://zenodo.org/records/19461436.

The model dashboard can be found at: https://tuur.github.io/files/dutchnlpllmoverview.html.

For any questions or suggestions, please contact: a.m.leeuwenberg-15@umcutrecht.nl

This repo's readme will be updated upon acceptance.
